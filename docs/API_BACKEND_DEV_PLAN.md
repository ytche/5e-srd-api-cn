# API 契约对齐后端开发计划

## 1. 目标与范围

本计划用于将 `5e-srd-api` 的 2014 版本实现与 `docs/API_CONTRACT.md` 完全对齐，重点包括：

- 双语字段（`*_en`）声明与返回一致性
- 字段命名/映射一致性（例如 `racial_traits -> traits`）
- 错误语义一致性（400/404/500）
- 子资源接口父资源存在性校验
- 契约回归测试体系

本计划仅覆盖后端 API，不涉及前端改造。

---

## 2. 现状总结

已完成：

- 全局错误默认状态修正为 `500`
- 部分 2014 实体补充 `*_en`
- `subrace` 详情映射为 `traits`
- `class/race/subrace` 若父资源不存在则返回 `404`
- 阶段 A-F 已落地（模型、语义、映射、契约测试、CI 守卫、PR 门禁）

仍需开发（下一轮）：

- 增加契约测试覆盖到关键集成路径（REST + GraphQL）
- 为图片等非核心资源定义单独契约（当前错误格式与主契约不一致）
- 契约变更发布流程标准化（release note 与破坏性变更声明）
- `2024` 资源契约化（暂缓，待后续排期）

## 3. 分阶段实施

## 阶段 A：模型层补齐（双语与契约字段）

目标：所有契约声明的字段在模型层显式存在。

任务：

1. 为以下模型补 `name_en`：
   - `alignment`, `condition`, `damageType`, `equipment`, `equipmentCategory`, `feature`, `language`, `magicItem`, `magicSchool`, `monster`, `proficiency`, `rule`, `ruleSection`, `skill`, `trait`, `weaponProperty`
2. 对包含 `desc` 的模型，补充 `desc_en`（类型与主字段对应，`string` 或 `string[]`）
3. 在 `Race` 模型补充：
   - `starting_proficiencies?: APIReference[]`
   - `starting_proficiency_options?: Choice`
4. 在 `Choice` 模型补充：
   - `desc_en?: string`

验收标准：

- TypeScript 编译通过
- 单测全绿
- 任意 5 个详情接口抽样可返回数据库中已有 `*_en` 字段

---

## 阶段 B：响应语义统一（404/500 与 404 JSON）

目标：错误行为与契约完全一致，可稳定被前端解析。

任务：

1. 新增统一 404 middleware（JSON）：`{ "error": "Not found" }`
2. 确保路由未命中、详情未命中、子资源父对象不存在三类场景均返回统一 JSON 404
3. 保持参数错误返回 `400 + details`
4. 保持未捕获服务端错误返回 `500 + { message }`

验收标准：

- `supertest` 覆盖以下场景并通过：
  - 未知路径 404
  - 不存在资源 index 404
  - 父资源不存在的子路径 404
  - 异常注入返回 500

---

## 阶段 C：契约映射与兼容策略

目标：数据命名与契约一致，同时兼容已有数据库命名。

任务：

1. 明确并实现映射清单（当前至少包括 `racial_traits -> traits`）
2. 为每个映射写单测，确保：
   - 契约字段存在
   - 旧字段不外露（除非明确需要兼容）
3. 在文档中维护“字段映射表”

验收标准：

- 映射接口契约测试全通过
- 文档映射表与实现一致

---

## 阶段 D：契约回归测试体系

目标：后续变更不会破坏 API 契约。

任务：

1. 新增 `src/tests/contract/` 测试目录
2. 覆盖：
   - 双语字段存在性
   - 列表响应结构 `{ count, results }`
   - 详情字段存在性（关键实体）
   - 错误语义（400/404/500）
3. 将契约测试纳入 CI（`npm run test:unit`）

验收标准：

- 新增契约测试稳定通过
- 任意一项契约偏离时测试可直接失败

## 5. 风险与缓解

1. 数据库字段不完整：
   - 缓解：保留字段可选，接口按“有则返回”
2. 旧前端依赖历史字段：
   - 缓解：映射期保持向后兼容窗口，发布说明标注
3. 错误码变更影响监控：
   - 缓解：同步更新告警规则与仪表盘
